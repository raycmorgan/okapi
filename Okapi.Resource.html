Okapi.Resource

Provides the functions to create the endpoint resources and also perform the actual HTTP(s) API calls.

delete(endpoint_name, path, options \\ [])

Creates an endpoint function that uses the DELETE HTTP verb.

The function that will be generated will be named endpoint_name. For example, given the following module.

defmodule Stripe do
  use Okapi

  resource Charge do
    delete :delete, "/charges/{id}"
  end
end

The function delete will be added to the Stripe.Charge module.

{:ok, result} = Stripe.Charge.delete(id: "ch_103KlI2eZvKYlo2Cb03HHP8s")

There will also be a ! version created that simply returns the result, but throws an error on failed requests.

result = Stripe.Charge.delete!(id: "ch_103KlI2eZvKYlo2Cb03HHP8s")

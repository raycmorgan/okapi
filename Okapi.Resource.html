get(endpoint_name, path, options // [])

Creates an endpoint function that uses the GET HTTP verb.

The function that will be generated will be named endpoint_name. For example, given the following module.

defmodule Stripe do
  use Okapi

  resource Charge do
    get :retrieve, "/charges/{id}"
  end
end

The function retrieve will be added to the Stripe.Charge module.

{:ok, charge} = Stripe.Charge.retrieve(id: "ch_103KlI2eZvKYlo2Cb03HHP8s")

There will also be a ! version created that simply returns the result, but throws an error on failed requests.

charge = Stripe.Charge.retrieve!(id: "ch_103KlI2eZvKYlo2Cb03HHP8s")

post(endpoint_name, path, options // [])

Creates an endpoint function that uses the POST HTTP verb.

The function that will be generated will be named endpoint_name. For example, given the following module.

defmodule Stripe do
  use Okapi

  resource Charge do
    post :create, "/charges/{id}"
  end
end

The function create will be added to the Stripe.Charge module.

{:ok, result} = Stripe.Charge.create(id: "ch_103KlI2eZvKYlo2Cb03HHP8s")

There will also be a ! version created that simply returns the result, but throws an error on failed requests.

result = Stripe.Charge.create!(id: "ch_103KlI2eZvKYlo2Cb03HHP8s")
